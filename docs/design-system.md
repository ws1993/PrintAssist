# 打印助手生产设计系统

## 已选方向

生产界面采用原型 A“明亮精密工作台”。它面向需要连续处理批量打印任务的熟练办公用户，优先保证信息扫描效率、状态可辨识度与参数可追溯性，而不是追求装饰性。

## 视觉语言

- 画布：冷白 `#F2F6FA`，可使用极淡 24px 网格辅助建立工作台秩序。
- 主表面：白色 `#FFFFFF`；次表面 `#F7F9FC`。
- 主文字：`#172033`；次文字 `#647086`；弱文字 `#8B95A7`。
- 品牌蓝：`#1557D0`；深蓝交互态 `#0B43AC`；浅蓝选中态 `#EAF1FF`。
- 成功、警告、失败分别使用 `#167354`、`#9B5D00`、`#B42333`，并始终配合文字，不只依赖颜色。
- 字体：Segoe UI Variable、Microsoft YaHei UI；编号和技术状态可使用 Consolas。
- 圆角保持 2-4px；以边线、网格和间距组织层级，避免大量悬浮圆角卡片。

## 布局和交互

- 顶栏 58px，左侧控制轨约 342px，右侧为弹性队列工作区。
- 文件入口与公共设置位于左侧；任务列表、批量操作和结果反馈位于右侧。
- 单文件设置采用右侧抽屉，保持队列上下文可见。
- 不支持的打印参数直接禁用并显示原因；查询失败显示“能力未知”，不能等同于“不支持”。
- 主要操作保持固定动词：“选择文件”“选择文件夹”“开始打印”“仅重试失败项”。
- 所有图标均保留文字标签；焦点态、键盘操作和状态播报属于生产验收范围。
