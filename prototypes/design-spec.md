# 打印助手：三方向原型设计说明

## 产品与场景

“打印助手”是一款仅面向 Windows 的批量文件打印控制工具。它服务于行政、人事、财务、教师、门店文员及其他需要频繁处理混合文件打印任务的人。用户可能从资源管理器右键菜单、“发送到”、拖放、选择单个文件或选择整个文件夹进入应用；这些入口都应将文件追加到当前队列，而不是替换既有任务。应用重启后队列清空，因此界面强调“当前批次”，不营造长期文档管理器的错觉。

## 原型目标

本轮不是生产界面开发，而是用三个真实可点击版本比较信息密度、视觉语言与操作节奏。三版必须使用同一组中文内容和相同能力边界，不能通过增减功能影响选择。每版均在 1440×900 桌面窗口下优先设计，并能在较窄窗口中保持可用。用户应能在一分钟内理解当前打印机、默认设置、文件数量、异常项和主要动作。

## 核心信息结构

1. 顶部应用区：产品名、当前批次状态、检查更新或帮助入口。
2. 文件入口：选择文件、选择文件夹和拖放区域；提示资源管理器右键入口也会追加文件。
3. 公共设置：系统打印机、颜色、单双面、长短边翻转、打印份数。打印机能力不支持的选项必须禁用并解释，不允许静默回退。
4. 待打印队列：文件名、类型、页数或待分析状态、设置继承状态、任务状态、移除与单独设置入口。
5. 单文件设置：页码范围支持“全部”和 `1,3,5-8`；可继承或覆盖颜色、单双面、翻转方式及份数。
6. 执行反馈：某个文件失败时其余文件继续；最终汇总成功、失败、跳过数量，并允许只重试失败项。

## 共同任务数据

- `2026年7月工资汇总.xlsx`：4 个工作表，使用公共设置。
- `新员工入职资料.pdf`：12 页，单独打印 `1,3,5-8`。
- `门店陈列照片.jpg`：1 页，强制彩色。
- `季度复盘会议.pptx`：26 页，等待分析。
- `打印说明.txt`：2 页，模拟失败状态以展示重试体验。

默认打印机为“HP Color LaserJet Pro M479”，状态在线，支持彩色和双面；“Microsoft Print to PDF”不支持设备双面能力；“仓库标签打印机”模拟离线。默认公共设置为黑白、双面长边、1 份、全部页。

## 三个设计方向

- 方向 A“明亮精密工作台”：冷静、清晰、高信息密度；用细线、网格和克制蓝色建立专业工具感，适合高频办公用户。
- 方向 B“温暖纸张工作室”：柔和、友好、有纸张与批次感；通过暖白、墨色、纸张层级降低复杂打印设置的压迫感。
- 方向 C“深色打印控制台”：深色但不使用通用霓虹科技模板；以设备面板、状态灯和高对比排版强调监控、可靠性和批处理进度。

## 交互和视觉约束

不得用无意义统计、夸张渐变、Emoji 图标或大量同质圆角卡片填充空间。图标采用 Lucide；原型允许通过 CDN 加载图标，但主要功能不能依赖图片。拖放、追加文件、切换打印机、单双面联动、打开单文件设置、保存页码表达式、开始打印、查看失败汇总和重试失败项都必须可点击演示。原型中的文件选择用模拟数据追加，避免浏览器安全限制妨碍评审。
